В данном образце создается новый список SharePoint (DonationList) и экземпляр
списка (DonationInstance) вместе с присоединенным приемником событий
(DonationDeleting).

Запуск образца

- Измените свойство «URL-адрес сайта» для проекта, указав действительный адрес
 сайта SharePoint в используемой системе разработки (т. е.
 http://<имя_компьютера>).

После создания списка к нему можно обращаться из меню быстрого запуска сайта
SharePoint. Создайте новую запись о пожертвовании на сумму, большую 0. Приемник
событий не позволит удалить ни один элемент пожертвования с положительной
суммой.

Примечание. Если запустить на одном компьютере оба образца — для VB и для C#, —
запуск второго проекта приведет к конфликтам имен с пакетом и экземпляром
списка.  Откроется диалоговое окно с предложением разрешить конфликты.
