Пример условных методов

Данный пример демонстрирует условные методы, обеспечивающие мощный механизм, с помощью которого вызовы методов могут быть включены или выключены в зависимости от задания символа.

Примечание по безопасности

Данный код примера предоставлен для иллюстрирования концепции и не должен использоваться в приложениях или веб-узлах, так как он не демонстрирует опыт написания безопасного кода. Microsoft не несет ответственность за случайные или косвенные повреждения, вызванные применением кода примера не по назначению.

Построение и выполнение примеров условных методов в Visual Studio.

  1. В Обозревателе решений щелкните правой кнопкой проект и выберите Свойства.

  2. Откройте папку Свойства конфигурации и щелкните Отладка.

  3. Установите для свойства Аргументы командной строки значение «A B C» (без кавычек).

  4. В папке Свойства конфигурации щелкните Построение.

  5. Измените значение свойства Константы условной компиляции (например, добавьте или удалите параметр DEBUG) и нажмите OK.

  6. В меню Отладка щелкните Запуск без отладки.

Построение и выполнение примеров условных методов из командной строки.

  • Для включения условного метода скомпилируйте и запустите программу примера путем ввода следующих данных в командную строку: 

    csc CondMethod.cs tracetest.cs /d:DEBUG
tracetest A B C
