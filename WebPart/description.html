В этом примере визуальной веб-части демонстрируется, как вывести список всех
семейств сайтов на локальном сервере SharePoint.  Для выбранного семейства
сайтов показываются статистические данные для каждого списка в этом семействе.

Для запуска примера необходимо выполнить следующие действия.

- Измените свойство "URL-адрес сайта" проекта на действительный сайт SharePoint
 на вашем компьютере разработки (например: http://<имя_компьютера>)

По запуску (F5) примера откроется окно браузера с домашней страницей вашего
сайта разработки.  В меню «Действия сайта» выберите команду «Дополнительные
параметры», а затем создайте страницу веб-частей.  После присвоения имени и
создания страницы добавьте на нее веб-часть.
